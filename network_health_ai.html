I need a SQL Server query that calculates the risk score trend for our Medicare Advantage members over the last 12 months, broken down by county. Include month-over-month change percentages.

ü§ñ

Here's a T-SQL query for your risk score trend analysis:

WITH MonthlyScores AS (
  SELECT 
    m.county,
    FORMAT(r.score_date, 'yyyy-MM') AS score_month,
    AVG(r.risk_score) AS avg_risk_score,
    COUNT(DISTINCT m.member_id) AS member_count
  FROM dbo.Members m
  JOIN dbo.RiskScores r ON m.member_id = r.member_id
  WHERE m.plan_type = 'Medicare Advantage'
    AND r.score_date >= DATEADD(month, -12, GETDATE())
  GROUP BY m.county, FORMAT(r.score_date, 'yyyy-MM')
)
SELECT 
  county,
  score_month,
  avg_risk_score,
  member_count,
  LAG(avg_risk_score) OVER (
    PARTITION BY county ORDER BY score_month
  ) AS prev_month_score,
  ROUND(
    (avg_risk_score - LAG(avg_risk_score) OVER (
      PARTITION BY county ORDER BY score_month
    )) / NULLIF(LAG(avg_risk_score) OVER (
      PARTITION BY county ORDER BY score_month
    ), 0) * 100, 2
  ) AS mom_change_pct
FROM MonthlyScores
ORDER BY county, score_month;

This uses a CTE for readability and LAG() window function for the month-over-month calculation. Want me to add YoY comparisons or adjust for any specific business rules?

Perfect. Now convert this to SAS for our VA reporting team.

ü§ñ

Here's the equivalent SAS code:

PROC SQL;
  CREATE TABLE work.monthly_scores AS
  SELECT 
    m.county,
    PUT(r.score_date, yymmn6.) AS score_month,
    AVG(r.risk_score) AS avg_risk_score,
    COUNT(DISTINCT m.member_id) AS member_count
  FROM lib.members m
  INNER JOIN lib.risk_scores r 
    ON m.member_id = r.member_id
  WHERE m.plan_type = 'Medicare Advantage'
    AND r.score_date >= INTNX('month', TODAY(), -12)
  GROUP BY m.county, CALCULATED score_month
  ORDER BY county, score_month;
QUIT;

/* Add MoM calculations */
DATA work.risk_trend;
  SET work.monthly_scores;
  BY county;
  prev_score = LAG(avg_risk_score);
  IF FIRST.county THEN prev_score = .;
  mom_change_pct = ROUND(
    (avg_risk_score - prev_score) / prev_score * 100, 0.01
  );
RUN;
